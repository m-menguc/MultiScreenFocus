\# Multi-Screen Focus 🖥️🌙



!\[Version](https://img.shields.io/badge/version-1.0.0-blue.svg)

!\[Platform](https://img.shields.io/badge/platform-Windows-lightgrey.svg)

!\[License](https://img.shields.io/badge/license-Freeware-green.svg)



\*\*Multi-Screen Focus\*\* is an advanced, lightweight Windows utility that automatically dims your secondary displays when your mouse is not actively on them. Designed to reduce eye strain, save energy, and block distractions, it features a modern UI, smart motion detection, and seamless background operation.



\## 📥 Download \& Install



Since this is a standalone utility, there is no need for installation or Python dependencies.



1\. Go to the \[Releases](../../releases/latest) page.

2\. Download the latest `MultiScreenFocus.exe`.

3\. Double-click to run. \*(Tip: Check the "Start with Windows" option in the app to make it run seamlessly in the background).\*



\## ✨ Features



\- \*\*Smart Motion Detection:\*\* Prevents the monitor from dimming if a video, movie, or any active movement is detected on the screen.

\- \*\*Independent Monitor Control:\*\* Set different timeouts and dimming ratios for each monitor individually through a modern accordion menu.

\- \*\*Live UI Updates:\*\* Adjust the dimming slider while the system is running and see the changes in real-time.

\- \*\*Ghost Overlay:\*\* The dimming layer does not block your mouse clicks and is completely invisible to screen recorders and the Windows Snipping Tool.

\- \*\*Focus Assist Bypass:\*\* Does not trigger Windows "Do Not Disturb" or full-screen gaming modes.

\- \*\*Auto-Start:\*\* Optionally start with Windows, quietly minimizing to the system tray.

\- \*\*Light \& Dark Mode:\*\* Toggle between themes instantly (☀️/🌙).

\- \*\*Bilingual Support:\*\* Fully supports English and Turkish.



\## ❓ FAQ



\*\*Q: Why doesn't the app open a window when I double-click it?\*\*

A: If you checked "Start with Windows (minimized to Tray)" previously, the application will quietly start in the background. Look for the blue feather icon in your system tray (next to the clock) and click "Show / Hide" to open the menu.



\*\*Q: Is the source code available?\*\*

A: Multi-Screen Focus is currently distributed as a closed-source freeware utility. The executable is securely packed and obfuscated to ensure integrity.

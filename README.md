Multi-Screen Focus is a lightweight Windows utility that automatically dims your secondary displays when your mouse is not on them. Reduce eye strain, save energy, and stay focused with a clean UI and seamless background operation.

<p align="center"><img width="520" height="150" alt="desktop_setup" src="https://github.com/user-attachments/assets/2d779101-02de-4406-8a12-95288a973da1" /></p>

## 📥 Download & Install

    No installation.
    
    Go to the Releases page.
    
    Download the latest MultiScreenFocus.exe.
    
    Double‑click to run.
    
    Tip: Enable “Start with Windows” for automatic background startup.

## ✨ Features

    Smart Motion Detection — Prevents dimming while videos or animations are active.

    Per‑Monitor Control — Individual timeout & dimming settings with accordion UI.

    Live UI Updates — Adjust dimming in real time.

    Ghost Overlay — Click‑through, invisible to screen recorders & Snipping Tool.

    Focus Assist Bypass — Does not trigger Windows DND or gaming mode.

    Auto‑Start — Optional tray‑only startup.

    Light & Dark Mode — Instant theme switching.

    English & Turkish Support
